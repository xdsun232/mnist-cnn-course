<!-- Slide 3: 核心知识点 - CNN架构 -->
## 核心知识点：CNN架构

### 网络结构

```
输入 [1,28,28] → Conv+ReLU+Pool → [32,13,13]
                → Conv+ReLU+Pool → [64,5,5]
                → 全连接层 → 128
                → 输出层 → 10
```

### 三大核心组件

| 组件 | 作用 |
|------|------|
| **卷积层** | 提取特征（边缘、纹理等） |
| **池化层** | 降采样，减少参数 |
| **全连接层** | 分类决策 |

<!-- Slide 4: 核心知识点 - 训练过程 -->
## 核心知识点：训练过程

### 训练循环

```python
for epoch in range(epochs):
    # 前向传播
    outputs = model(images)
    loss = criterion(outputs, labels)

    # 反向传播
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

### 关键概念

- **Loss**: 预测与真实的差距
- **学习率**: 参数更新步长
- **Batch**: 每次处理的样本数
- **Epoch**: 完整遍历数据集一次

<!-- Slide 5: 核心知识点 - 模型优化 -->
## 核心知识点：模型优化

### 优化技巧

| 技巧 | 效果 |
|------|------|
| **数据增强** | 旋转、平移图像，增加多样性 |
| **Dropout** | 随机丢弃神经元，防止过拟合 |
| **BatchNorm** | 归一化层输入，加速训练 |
| **学习率调度** | 后期降低学习率，提高精度 |

### 防止过拟合
- 训练准确率 >> 测试准确率 → 过拟合
- 解决：Dropout、数据增强、早停
